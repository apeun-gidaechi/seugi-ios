## 🏃‍♂️ Run Project

### Files

- Add `./Projects/App/Seugi-iOS/Resources/GoogleService-Info.plist`

### Project
- Tusit version: 3.35.1
- [Use mise to install](https://docs.tuist.io/guide/introduction/installation)


```bash
mise exec -- tuist fetch
make
```
